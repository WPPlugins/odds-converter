== Description ==

Odds converter will convert odds between decimal (separated with .), fractional (example: 1/2) and american (example: -200 or 120).

This section describes how to install the plugin and get it working.

1. Install oddsconverter.zip throught 'Plugins' menu and activate it.
2. Edit title and description throught 'Appearance' -> 'Widgets' (find Modulout - Odds converter widget)
